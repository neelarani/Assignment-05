What is the Document Object Model (DOM)?

The Document Object Model(DOM) is a programming interface for HTML or XML document. It represents each element like tags, attributes, and text as an object, enabling easy interaction and control.

With the DOM, you can use javascript to add. remove, modify, or update elements, facilitating interactivity and dynamic content on web pages.

How do you select an element from the DOM?

getElementById: used to select an element with a specific ID.
Example: const element = document.getElementById('elementId');

getElementsByClassName: used to select all element with a specific class name.
Example: const element = document.getElementsByClassName('className');

getElementsByTagName: used to select all element with a specific tag name.
Example: const element = document.getElementsByClassName('tagName');

querySelector: used to select the first element that matches a css selector.
Example: const element = document.querySelector('.className');

querySelectorAll: used to select all element that matches a css selector.
Example: const element = document.querySelectorAll('div');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where an event listener is added to a parent element, which can capture events form child element.


Benefits:

  • One listener can manage events for many child elements.
  • No need for separate listeners when new elements are added.
  • Fewer listeners improve performance.
